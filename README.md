# 🚀 Flowlytics - User Behavior Analytics Dashboard

A SaaS-style analytics simulation project that tracks user interactions such as clicks, scroll depth, page views, and form submissions using JavaScript event tracking.

---

## 📊 Features
- Multi-page website (Home, Features, Dashboard)
- Click tracking using JavaScript
- Scroll depth tracking
- Form submission tracking
- Dashboard with charts (Chart.js)
- SaaS-style modern UI

---

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript
- Chart.js

---

## 🎯 Project Goal
To simulate Google Analytics + Google Tag Manager style event tracking in a simple front-end project for learning web analytics concepts.
